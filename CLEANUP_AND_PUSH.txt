================================================================================
                    CLEANUP AND PUSH TO GITHUB
================================================================================

Run these commands in PowerShell from the energy-analytics-system directory:

================================================================================
STEP 1: REMOVE UNWANTED FILES
================================================================================

# Remove temporary and redundant files
rm deployment_info.json
rm report.txt
rm NEXT_STEP.txt
rm RUN_THIS_NOW.txt
rm TEST_LAMBDA_NOW.txt
rm FINAL_STEPS.txt
rm PUSH_TO_GITHUB_NOW.txt
rm RUN_THIS_TO_PUSH_GITHUB.txt
rm START_HERE_FINAL.md

# Note: data/output/energy_data.csv is already excluded by .gitignore

================================================================================
STEP 2: CHECK GIT STATUS
================================================================================

git status

# This will show:
# - New files to be added
# - Modified files
# - Deleted files

================================================================================
STEP 3: ADD ALL CHANGES
================================================================================

git add .

# This adds all new and modified files
# Deleted files will be staged for removal

================================================================================
STEP 4: COMMIT CHANGES
================================================================================

git commit -m "Add comprehensive documentation and cleanup temporary files"

================================================================================
STEP 5: PUSH TO GITHUB
================================================================================

git push origin main

# If this is your first push or you need to set upstream:
git push -u origin main

================================================================================
ALTERNATIVE: ALL IN ONE COMMAND
================================================================================

# Run all steps at once:
rm deployment_info.json, report.txt, NEXT_STEP.txt, RUN_THIS_NOW.txt, TEST_LAMBDA_NOW.txt, FINAL_STEPS.txt, PUSH_TO_GITHUB_NOW.txt, RUN_THIS_TO_PUSH_GITHUB.txt, START_HERE_FINAL.md; git add .; git commit -m "Add comprehensive documentation and cleanup temporary files"; git push origin main
